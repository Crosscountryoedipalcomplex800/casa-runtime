## 🛑 Troubleshooting

### The Installer Won’t Run

- Confirm you downloaded the `.exe` file, not a compressed folder.
- Right-click the file and select "Run as administrator" to avoid permission issues.
- Disable antivirus temporarily if it blocks the installer, then re-enable afterward.

### casa-runtime Won’t Launch or Crashes

- Check your Windows version meets the minimum requirements.
- Restart your PC and try again.
- Ensure you have enough free RAM and disk space.
- Reinstall casa-runtime from the latest release.

### Agents Do Not Follow Rules

- Verify you enabled execution controls in the session setup.
- Double-check the rules you applied for accuracy.
- Make sure audit logs record what you expect. Logs can help spot misconfigurations.
